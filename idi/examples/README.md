# IDI Examples & Tutorials

This folder contains **production-ready** Tau Language intelligent agents with full documentation and tutorials.

## Purpose

Examples here are:
- ✅ Fully tested and verified
- ✅ Well-documented with tutorials
- ✅ Ready for teaching and demonstration
- ✅ Following IDI best practices

## Structure

Each example includes:

```
examples/
├── example_name/
│   ├── spec.tau              # Tau specification
│   ├── README.md             # Tutorial and explanation
│   ├── inputs/               # Example inputs
│   ├── outputs/              # Expected outputs
│   ├── tests/                # Test suite
│   └── tutorial.md           # Step-by-step guide
```

## Adding New Examples

When moving an agent from `practice/` to `examples/`:

1. Ensure all tests pass
2. Write comprehensive README.md
3. Create tutorial.md with step-by-step guide
4. Include example inputs/outputs
5. Document any IDI patterns used
6. Add to this README's index
